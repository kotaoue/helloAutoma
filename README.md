# helloAutoma

A collection of [Automa](https://chromewebstore.google.com/detail/automa/infppggnoaenmfagbfknfkancpbljcca) workflows.

## Getting started

1. Install the [Automa](https://chromewebstore.google.com/detail/automa/infppggnoaenmfagbfknfkancpbljcca) browser extension.
2. Browse the [`workflows/`](workflows/) directory and open the JSON file you want.
3. Use the **[GitHub → Automa Import](workflows/github-import/github-import.json)** workflow to download a workflow JSON from any GitHub page and import it into Automa with minimal steps.

See [`workflows/README.md`](workflows/README.md) for details on each workflow.
